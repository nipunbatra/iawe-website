Unique features in India- Unreliable grid

Learning

  1. System Design: System should be capable of resuming in same state as it was before outage
  2. Inferences: Voltage needs to be measured (in addition to current) and cannot be assumed constant (applications in NILM)

Internet

Unique features in India- Unreliable network

Learning

  1. System Design: System should account for unreliable internet
  2. We did the same by using "Local storage"- Each RPi would store data locally and periodically upload data outside
  3. As a result, we lost no data due to unreliable network

Unique features in India- Load specifics

  1. Decentralized control: Room level ACs, bathroom level water heating
  2. About 60% of total load can be attributed to AC (in summer) and water heater (in winter)
  3. Fairly easy to disaggregate high percentage of total energy
